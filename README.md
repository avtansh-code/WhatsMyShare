# What's My Share

A bill-splitting application for easy and efficient expense management. This application is designed to simplify tracking shared expenses with friends, family, and colleagues.

## Core Features

*   **User Management:** Simple and secure user registration, authentication (including OAuth with Google), and profile management.
*   **Group & Friend Management:** Create expense groups, add friends, and track individual expenses. Includes a "Simplify Debt" feature to minimize the number of transactions required for settlement.
*   **Flexible Expense Splitting:**
    *   Split expenses equally.
    *   Split by exact amounts.
    *   Split by percentage or ratio.
    *   Support for multiple payers.
*   **Seamless Settlements:** Record payments and settle debts with ease.
*   **Real-time Notifications:** Stay updated with push notifications for group activities, new expenses, and settlements.
*   **Offline Support:** Add expenses even when you're offline. The app will sync automatically when you reconnect.

## Technology Stack

*   **Frontend:** Flutter
*   **Backend:** Google Cloud Platform (GCP)
